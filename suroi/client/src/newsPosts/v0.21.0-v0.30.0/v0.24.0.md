2월 14일은 우리의 주요 영감이었던 Surviv.io가 영구적으로 폐쇄된 지 2년이 되는 날이었습니다. 이를 기념하여 오랫동안 기다려온 게임플레이 변경 사항과 새로운 구조물, 두 개의 새로운 총을 준비했습니다. 게임도 클래식 모드로 전환되었습니다.

가스 구역이 처음부터 완전히 리워크되었습니다. 구역이 이제 더 공정하고 더 잘 배치되었으며, 흥미로운 새로운 최종 링이 구현되었습니다. 기본 이동 속도를 높였고, 아드레날린이 더 많은 속도와 더 적은 체력을 제공하도록 리워크하면서 낮은 수준의 아드레날린에서 더 큰 혜택을 제공합니다. 하베스트에서의 테스트 인기로 인해 2배율 스코프가 기본 스코프가 되었고, 모든 스코프 변형에 대한 새로운 아트도 추가했습니다.

**MP5k**는 다른 중급 무기가 스폰될 수 있는 거의 모든 곳에서 발견되는 새로운 버스트 기관권총입니다. 인상적인 DPS와 제어 가능한 퍼짐을 자랑하는 이 중거리 9mm 총은 듀얼과 싱글 모두에서 뛰어납니다.

**메모리얼**은 우리 전임자에게 바치는 엄숙한 기념물입니다. 쓰러진 자들에게 경의를 표하면 마지막 선물이 기다리고 있을 수 있습니다... **PK-61**. 전설적인 PKM의 전신인 이 대용량 기관총은 낮은 발사 속도를 대가로 뛰어난 정확도와 피해, 특히 장애물에 대한 피해를 특징으로 합니다. 메모리얼에는 중간 피해를 가진 빠른 근거리 근접무기 **쿠크리**도 추가되었습니다.

여러 무기, 폭발물, 오브젝트 체력 등 게임의 대부분에 다양한 수정, 업데이트, 리밸런스가 있었습니다.

# Suroi v0.24.0

## 새 기능
- **새 구조물!** surviv.io에 대한 헌정인 메모리얼 추가.
- **새 총!**
  - MP5K: 맵 전체에서 발견되는 9mm 버스트 권총.
  - PK-61: 새 메모리얼에서만 발견되는 강력한 7.62mm 경기관총.
- 실험적 화면 녹화 기능 추가.
  - 향후 더 부드러운 재생과 더 작은 파일 크기를 허용하는 패킷 기반 녹화를 구현할 계획입니다.
- 모바일 조이스틱에 대한 사용자 정의 옵션 추가.
- 2개의 새 이모트 추가.
- 오세아니아 서버 추가.

## 밸런스 변경
- 가스 리밸런스. 게임이 이제 5분 반에서 6분 반으로 길어지고, 가스가 덜 공격적이며 더 흥미로운 최종 서클.
- 플레이어 이동 속도 증가.
- 아드레날린이 이제 낮은 수준에서 더 효과적이고, 높은 수준에서 약간 덜 효과적.
- 플레이어가 이제 초당 1의 속도로 자연 체력 재생.
- 붕대가 이제 20 체력 회복, 15에서 증가.
- 진통제 사용 시간이 5초에서 4초로 감소.
- 2배율이 이제 모든 모드에서 기본 스코프.
- CZ-600을 제외한 모든 저격총이 이제 스코프와 함께 스폰.
  - 모신나강, RG 스카웃, Mk-18 묠니르: 4배율
  - 탱고 51, VKS 비흘롭: 8배율
  - L115A1: 16배율
- 프라이팬 크기 감소.
- M590M, BLR 556, SAF-200, 네게브 버프.
- 벡터 사거리 너프.
- HP-18이 이제 장애물에 대해 두 배로 효과적 (장애물 배율 0.5 -> 1).
- 상자, 쓰레기봉투, 덤프스터, 쓰레기통, 대형 정유소 배럴을 포함한 여러 장애물의 체력 감소.
- 대부분의 폭발에 대한 장애물 피해 배율을 2배로 증가.
- 덤프스터 루트 테이블 조정.

## 일반 변경
- 스코프 스프라이트 재디자인.
- 15배율 스코프가 16배율로 이름 변경.
- 14개 총의 월드 스프라이트 업데이트.
- 소음기 총의 총알 불투명도 감소.
- 뉴스 페이지 코드 재작성 및 페이지네이션 추가.
- 초록 집에 Model 37과 SKS 총기 거치대 추가.
- 프라이팬 스토브 스폰 확률 5%에서 3%로 감소.
- 정유소에서 스토너 63 스폰 확률 1%에서 5%로 증가.
- 팀 모드에서 플레이어 이름이 이제 플레이어 표시기 색상과 일치.
- 연막 수류탄 성능 최적화.
- 새 근접무기 애니메이션 추가. 다음 업데이트에서 더 개선될 예정.
- 폭격 무기고의 부서지는 벽에 균열 추가하여 더 명확하게 표시.
- SAF-200과 VSS 발사 소리에서 탄피 소리 제거.
- 모바일 UI 조정.
- DEagle 루트 이미지 업데이트.
- 네게브 킬피드 이미지 업데이트.
- 새 K-bar 전환 소리.
- 스토너 63 재장전 소리에서 배경 노이즈 제거.
- 팀 모드에서 사망 화면이 이제 팀 전원이 죽지 않은 한 "팀이 전멸했습니다." 대신 "사망했습니다." 표시.
- "your did it" 뱃지가 이제 0킬 0피해로 승리할 때만 표시.
- 다양한 번역 업데이트.
- 서버 코드 리팩토링, 더 쉬운 확장성과 서버 호스터를 위한 더 쉬운 배포.

## 버그 수정
- 플레이어가 너무 가깝게 스폰되는 문제 수정.
- 뉴스와 규칙 페이지가 스크롤되지 않는 문제 수정.
- 이동 부드럽게 비활성화 시 퍽이 표시되지 않는 문제 수정.
- 관전 시 퍽이 업데이트되지 않는 문제 수정.
- 강 장애물 생성 관련 일부 문제 수정.
- 게임 오버 통계 번역 누락 수정.
- 프라이팬 충돌 버그 수정.
- 헛간 스폰 히트박스에 인접한 건초 창고가 포함되지 않는 문제 수정.
- 관전 시 플레이어 간 전환할 때 행동 타이머가 숨겨지는 문제 수정.
- 듀얼 건에서 전환 소리가 재생되지 않는 문제 수정.
- 신고 관련 일부 문제 수정.
- 향상된 그래픽 총 가스 파티클 버그 수정.
- 플럼킨 벙커 믹서가 너무 느리게 회전하는 문제 수정.
- 모바일에서 규칙 및 튜토리얼 닫기 버튼이 너무 넓은 문제 수정.
