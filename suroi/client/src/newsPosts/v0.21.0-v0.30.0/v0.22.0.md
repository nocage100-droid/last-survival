겨울이 섬에 도착하여 시즌의 축제 분위기와 두 개의 새로운 총을 가져왔습니다.

일반 모드가 이제 모든 에셋에 눈 덮인 리스킨과 무기를 드롭하는 새 선물을 가지며, 많은 것들이 야외 크리스마스 트리 아래 배치되어 있습니다. **빙벽 곡괭이**도 돌아왔으며 사용할 새로운 고가치 **얼어붙은 상자**와 함께, 총알과 특정 다른 근접무기도 작동합니다... 결국.

하베스트 한정으로, **BLR**은 5발 탄창, 높은 피해, 중간 정도의 전체 스탯을 가진 새로운 비공통 **5.56mm DMR**로, 무작위 스팸 대신 신중한 조준에 보상합니다.

**RGS 저격소총**이 Red Tugboat에서 CZ-600을 대체했으며, **10발** 탄창과 같은 구경으로 챔버되면서 전임자보다 전반적으로 개선된 스탯을 자랑하지만 클래식 모드에서는 다른 곳에서 찾을 수 없습니다. 하베스트 모드에서는 에어드랍, 강과 바이킹 상자, 총기 보관함에서도 찾을 수 있습니다.

**루이스 건**이 대규모 개편을 받아 더 빠른 발사 속도, 감소된 이동 페널티, 전투에서 덜 고통스러운 더 공정한 크기의 스프라이트를 자랑합니다.

주간 로테이션 모드는 **12월 15일 일요일 오후 8시 EST**에 시작됩니다. 모드 로테이션은 **필요에 따라 그리고 커뮤니티 피드백에 대응하여 향후 조정될 예정입니다**.

# Suroi v0.22.0

## 새 기능
- **겨울 맵!** 눈 덮인 건물과 장애물이 있는 일반 모드의 리스킨 변형.
- **새 구조물!** 겨울 맵 한정 크리스마스 캠프 추가.
- **새 총!**
  - BLR: 하베스트 모드 한정 5.56mm 탄약을 사용하는 높은 피해지만 낮은 용량의 DMR.
  - RGS: 높은 용량과 발사 속도를 특징으로 하는 5.56mm 탄약을 사용하는 저격소총.
- **새 근접무기!** 겨울 맵 한정 빙벽 곡괭이 추가.
- 창고와 파란 집 금고의 새로운 레이아웃 변형.
- 겨울 맵 에어드랍 한정 라이트 초코 스킨 추가.

## 변경 사항
- 루이스 건이 더 높은 피해(16 -> 16.5), 낮은 발사 지연(120 -> 115), 감소된 속도 페널티(0.65x -> 0.7x)로 리밸런스되었으며 감소된 장애물 배율(2.5 -> 2)을 대가로 함.
- 벡터 피해 감소(10 -> 9).
- 탱고 51, CZ-75A, 플루스, Mini-14, 모신나강, Model 37, HP-18, M16A4의 루트 이미지 업데이트.
- Model 37 월드 이미지 업데이트.
- CZ-600이 이제 일반 맵에서 더 흔하고 에어드랍에서 제거됨.
- 이모트와 팀 핑에 쿨다운 추가, 스팸 방지.
- 일반 맵에서 대부분의 큰 나무 제거.
- VSS를 VSS 빈토레츠로, PP-19를 PP-19 비티아즈로 이름 변경.
- 자작나무 스프라이트 조정.
- 새 낫 휘두르기 애니메이션.

## 버그 수정
- 무기고 금고의 천장 가시성 문제 수정.
- 잘못 스케일된 아이템 요청 이모트 수정.
- 항구의 두 덤프스터 회전 수정.
- 과대한 DEagle 월드 이미지 수정.
- 본부 소파의 Z 인덱스 문제 수정.
