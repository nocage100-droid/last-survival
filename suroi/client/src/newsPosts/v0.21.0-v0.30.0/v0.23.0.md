겨울 모드가 끝나고 주간 모드 로테이션이 시작되었습니다. 파란 집이 RSh-12가 들어있는 벙커가 있는 변형을 받았으며, 본부 내부의 오래된 무기 스폰을 대체합니다. 게임 오버 화면에 팀 통계도 추가되었습니다.

업데이트된 회복 아이템 텍스처, 새로운 컨테이너 색상, 지난 업데이트에서 제거된 대부분의 이모트 복원을 포함하여 게임 전반에 걸쳐 수많은 조정과 수정이 이루어졌습니다.
***
# Suroi v0.23.0

## 새 기능
- **모드 전환!** 게임이 이제 매주 하베스트와 클래식 모드 사이를 전환합니다.
- **새 벙커!** 지하실이 있는 파란 집 변형 추가.
- **팀 통계!** 이제 게임 오버 화면에서 팀의 모든 사람의 통계가 표시됩니다.
- 개선된 디버그 표시 추가, "네트워크 그래프 스타일 조정" 키바인드로 접근 가능.
- 이전 업데이트에서 제거된 대부분의 이모트 다시 추가, 일부 이모트 조정, 새 이모트 1개 추가.

## 변경 사항
- 회복 아이템의 더 좋은 텍스처.
- 수류탄, 근접무기, AEGIS, Flint, 탄약 상자, 박스의 텍스처 개선.
- 텍스처 로딩 방식 변경, 로딩 시간 개선.
- 메인 메뉴 소셜 개선.
- 헛간 레이아웃 개선.
- USAS-12와 M590M 너프.
- 네게브 리밸런스.
- 본부에서 RSh-12 제거, 이제 파란 집 지하에서 발견.
- 하베스트 모드 에어드랍에서 SR-25 제거.
- 듀얼 M1895가 이제 28발 대신 42발의 7.62mm와 함께 스폰.
- 하베스트 모드의 서류가방이 이제 M1 Garand를 포함할 확률.
- 새 CZ-600과 MP40 소리.
- Micro Uzi와 MP40 소리 조정.
- 루이스 건과 USAS-12 킬피드 이미지 조정.
- 루이스 건, USAS-12, M1895 루트 이미지 조정.
- RGS를 RG 스카웃으로, 네게브를 네게브 SF로, MG-36을 MG36으로 이름 변경.
- 컨테이너 색상 조정.
- 이층 침대와 구명조끼 텍스처 개선.
- 벙커 진입/퇴장 시 환경음이 갑자기 멈추는 대신 페이드 인/아웃.
- 일반적으로 미니맵에 숨겨진 장애물(예: Flint 상자)이 건물의 일부로 생성될 때 미니맵에 표시.
- 키바인드가 이제 더 관대하고 잘못된 키바인드를 자동 수정.
- 다양한 번역 업데이트.

## 버그 수정
- 폭발 관련 버그 수정.
- "5초 후 JoinPacket을 받지 못함" 오류 수정.
- 게임이 솔로/듀오/스쿼드 간 전환될 때 페이지가 새로고침되는 문제 수정.
- 하베스트 모드에서 플레이어에게서 2배율 스코프가 드롭되는 문제 수정.
- 강의 장애물 분포 불량 수정.
- 플럼킨 벙커의 금고 문 위로 수류탄을 던질 수 있는 문제 수정.
- 게임에서 폰트가 가끔 로드되지 않는 문제 수정.
- 번역/언어 퍼센티지가 100%를 초과하는 문제 수정.
