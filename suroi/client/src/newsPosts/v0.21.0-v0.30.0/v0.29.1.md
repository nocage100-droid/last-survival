# Suroi v0.29.1
- **새 총!** 강력한 5.45mm 리볼버 **TTK 에사울** 추가.
- 헌티드 모드로 전환.
- Node.js에서 Bun 런타임으로, nginx에서 HAProxy로 전환하여 성능 향상.
- 반사 탄환 퍽이 더 이상 총알을 플레이어에게서 튕기지 않음.
- 다른 플레이어에게 플레이어 크기가 보이지 않는 문제 수정 (예: 로우 프로파일 및 과체중 퍽).
- 죽은 플레이어가 감염되는 문제 수정.
- 듀얼 MP5k에 확장 탄창이 없는 문제 수정.
- S.E.E.D. 투척물의 킬피드 이미지 추가.
